# Smart Convention Hall Booking and Management System Using Artificial Intelligence

This is a first-year final project built with Flask. It helps users book convention halls, get AI-style hall recommendations, estimate event budgets, ask a simple chatbot, manage bookings, and download booking reports as PDF files.

## Features

- User registration and login
- Hall booking with date conflict checking
- View, edit, search and delete bookings
- AI-style hall recommendation based on event type and guests
- Budget estimator for hall, food and decoration
- Rule-based chatbot for common customer questions
- Admin dashboard and user list
- PDF booking report download

## How to Run

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Start the application:

```bash
python app.py
```

3. Open this URL in a browser:

```text
http://127.0.0.1:5000
```
